# Agentic Developer Portal

A developer portal where you can see all your repositories in one place and run AI-powered tasks on them with a single click.

## What it does

- **Browse your repos** — see all repositories in a sidebar with search and filters by language, visibility, and CI status.
- **Repo health at a glance** — each repo shows a health score, open issues, vulnerabilities, dependencies, contributors, and CI status.
- **Run AI agent tasks** — pick a repo and trigger tasks like:
  - Create a Pull Request
  - Refactor code
  - Upgrade dependencies
  - Run tests
  - Security scan
  - Code review
- **Live log streaming** — watch the agent work in real time with a log output panel.
- **Task details** — once a task finishes, open a details modal to see the full summary and results.

## Tech stack

- React + TypeScript
- Vite
- Tailwind CSS
- Lucide icons

## Getting started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.
